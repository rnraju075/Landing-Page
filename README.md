A modern, high-performance landing page built with React 19 and Vite, styled using Tailwind CSS v4.
The project focuses on smooth animations, responsive layouts, and fast development using the latest frontend tooling.
Motion effects are powered by Motion, interactive sliders by Swiper, and icons via React Icons.
TypeScript ensures type safety, while ESLint maintains code quality.
